# stock-analysis-pro 使用指南

A股全维度分析 + ETF 期权分析工具 — 通过 Hermes Agent 对话调用，或命令行直接执行。

---

## 🚀 快速上手

对 Agent 说：

| 你想做什么 | 怎么说 |
|-----------|--------|
| 每日复盘 | `复盘` 或 `今天市场怎么样` |
| 期权扫描 | `扫描期权` 或 `看看期权机会` |
| 分析一只股票 | `分析 600519` 或 `分析一下贵州茅台` |
| 看看今天大盘 | `市场概览` 或 `今天行情怎么样` |
| 扫描热门概念 | `概念扫描` 或 `今天哪些板块强` |
| 批量分析自选股 | `分析自选股` |

Agent 会自动调用本工具并给你解读结果。

---

## 📱 五大核心功能

### 1. 每日复盘（新增 v3.0）

**一句话触发：** `复盘` 或 `今天市场怎么样`

输出包含：
- **📊 指数概览** — 上证/深证/创业板/科创50 实时行情
- **📈 涨跌统计** — 涨跌家数、涨停跌停数、赚钱效应分析
- **💰 概念资金流** — 资金净流入 Top10 概念板块
- **💼 持仓跟踪** — 当日持仓盈亏、涨跌、风险提示
- **👁️ 自选股监控** — 自选股列表及表现

数据来源：腾讯行情 + 东财 push2 API + Playwright 涨跌停数据

### 2. ETF 期权扫描（新增 v3.0）

**一句话触发：** `扫描期权` 或 `看看期权机会`

输出包含：
- **🎯 卖方机会** — IV > HV 的虚值期权，按时间价值衰减速度排序
- **🎲 买方机会** — IV < HV 的平值/浅虚值期权，按性价比排序
- **⚠️ 风险指标** — Delta/Gamma/Theta/Vega 完整希腊字母分析

支持多标的扫描（50ETF、300ETF、500ETF 等），自动过滤流动性差的合约。

数据来源：新浪期权频道 + akshare 期权数据 + 腾讯标的行情

### 3. 个股深度分析

**一句话触发：** `分析 [股票代码或名称]`

输出包含：
- **📊 大盘环境** — 上证/深证/创业板实时涨跌，个股相对强度
- **🏢 公司概况** — 主营业务、产品类型、实控人
- **🎯 综合评分** — 四维度打分(-100~+100)，7级评级(强看多→强看空)
- **📈 技术面** — MA均线、MACD、KDJ、RSI、BOLL、量价关系、均线排列
- **💰 基本面** — ROE、毛利率、负债率、营收/净利增速、机构一致预期、分红历史
- **🏦 资金面** — 成交额统计、北向持股(历史)、大股东变动
- **📰 舆情** — 股吧热帖情绪、互动易问答、最新新闻、分析师评级
- **🔄 历史对比** — 与上次分析对比(价格/评分/评级变化)

**评分体系：**

| 分数 | 评级 | 含义 |
|------|------|------|
| +60~+100 | 强看多 🟢🟢 | 多维度共振看多 |
| +30~+59 | 看多 🟢 | 整体偏多 |
| +10~+29 | 偏多 🟢 | 轻度看多 |
| -9~+9 | 中性 ⚪ | 多空均衡 |
| -29~-10 | 偏空 🔴 | 轻度看空 |
| -59~-30 | 看空 🔴 | 整体偏空 |
| -100~-60 | 强看空 🔴🔴 | 多维度共振看空 |

### 2. 概念板块扫描

**一句话触发：** `概念扫描` 或 `今天什么板块强`

自动完成 4 步分析：
1. 📋 **排行** — 按资金流入排序获取概念板块，过滤非行业概念（风格/市值/地域类）
2. 📈 **趋势** — 分析100只成分股涨跌分布，判断板块阶段(金叉启动/主升浪/上升/走弱)
3. 📰 **新闻** — 搜索板块相关最新消息
4. 💡 **机会** — 综合评分(赚钱效应/介入时机/资金强度/板块宽度)，找出领涨个股

输出示例：
```
【1】半导体  +3.25%  ⭐ 重点关注(78分)
    趋势: 🔥 金叉启动 — 刚启动32% + 上涨面68%
    📰 [2026-06-14] 芯片国产替代加速 (证券时报)
    💡 成分股: 涨68/平5/跌27  均涨+2.8%
    🏆 领涨: 中芯国际(SH688981) +8.12%
```

### 3. 宏观市场概览

**一句话触发：** `市场概览` 或 `今天大盘怎么样`

输出包含：
- 🌍 **国际环境** — 美债10Y、美联储利率、黄金/白银/原油
- 🇨🇳 **国内经济** — CPI、PMI、M2、LPR
- 🔥 **赚钱效应** — 涨停数、连板高度、热门方向
- 🎯 **综合研判** — 操作建议(积极看多/谨慎/防守)

## 💬 进阶用法


### 指定输出格式

对 Agent 说：
- `给我 600519 的 JSON 摘要` → 输出紧凑 JSON(~200 token)
- `600519 简要分析` → 只输出核心结论

### 组合使用

一次对话中可以连续问：
```
用户: 今天大盘怎么样？
Agent: [输出市场概览]

用户: 哪些概念板块强？
Agent: [输出概念扫描]

用户: 分析一下排名第一的那个概念里的龙头股
Agent: [对龙头股做个股深度分析]
```

## 📊 输出模式对照

| 模式 | 触发 | Token 消耗 | 适用场景 |
|------|------|-----------|---------|
| 完整报告 | `分析 600519` | ~2000 | 深度研究 |
| HTML报告 | `--html` | — | 可视化报告（暗色主题，移动端适配） |
| JSON 摘要 | `--summary` | ~200 | Agent 快速判断 |
| 简要模式 | `--brief` | ~50 | 只看价格/PE |

**HTML报告说明：**

三大分析（个股/概念/市场）均支持 `--html` 参数输出完整HTML报告：
- 暗色主题 + CSS变量 + 移动端适配
- 6个可复用组件（指标网格/信号标签/评分仪表盘/进度条/数据表格/折叠面板）
- 适合在微信内置浏览器查看

```bash
# 生成HTML报告
python3 core/cli.py analyze 600519 --html   # → output/stock_600519.html
python3 core/cli.py concept --html          # → output/concept.html
python3 core/cli.py market --html           # → output/market.html
```

## ⚠️ 已知限制

| 限制 | 原因 | 影响 |
|------|------|------|
| 个股主力资金流不可用 | 东财 push2his 接口被封 | 资金面评分缺少主力数据 |
| 北向持股数据过时 | 港交所 2024-08 停止披露 | 北向评分自动降权(×0.3) |
| 融资融券不可用 | 接口不稳定 | 资金面缺少两融数据 |
| 大宗交易不可用 | 无可用数据源 | 无法分析大宗 |
| 行业估值对比缺失 | 行业均值 API 被封 | 只能看绝对估值，无法横向对比 |
| 互动易已接入 | — | 股吧+互动易+新闻+分析师评级均已可用 |

**北向数据说明：** 港交所自 2024 年 8 月起停止披露个股级北向持股数据。系统会自动检测数据新鲜度，过时数据在评分中降权处理，信号旁会标注 ⚠过时。

## 🔧 技术细节

- **项目位置：** `/tmp/stock-analysis-pro/`
- **数据源：** 腾讯行情 + 新浪 K 线 + 东财 F10/新闻/研报/互动易/push2概念(Playwright拦截) + akshare(财务/北向/涨停)
- **缓存策略：** 行情缓存 1 小时，概念扫描缓存 10 分钟
- **评分权重：** 技术/基本面/资金/舆情 各 25%（可在 `config/config.yaml` 调整）
- **代理设置：** akshare 接口需走代理（已自动配置）
- **配置文件：** `config/config.yaml`（东方财富Cookie在此管理）
- **GitHub：** `bobby1129/stock-analysis-pro`
